# 📊 How to Find Test Reports

## Quick Answer

After running tests, your HTML report is saved in **TWO ways**:

### Method 1: View/Download Buttons (Easiest!)

After clicking "Build & Run Tests":
1. Look for **"View Report"** and **"Download Report"** buttons
2. They appear right below the test results title
3. Click **"View Report"** → Opens in new browser tab
4. Click **"Download Report"** → Downloads HTML file

### Method 2: Reports Page

1. Go to http://localhost:3000/**reports.html**
2. See ALL your test reports in one place
3. Each report shows:
   - Test directory name
   - Creation date
   - File size
   - View and Download buttons

### 1. Generate Tests
```
http://localhost:3000/generate.html
↓
Select project
↓

↓

```

```
Click "Build & Run Tests"
↓
Watch build output
```

### 3. View Report (Choose ONE)

**Option A: Click "View Report" button**

- Shows formatted HTML report
- Professional grey/white design

**Option B: Click "Download Report" button**
- Downloads as: `cpputest-report-{directory}.html`

- http://localhost:3000/reports.html
- See ALL reports
- Click any report's buttons

**Option D: Find on Disk**
- Navigate to `generated_tests/`
- Find your test directory (e.g., `tests_20260209_210000`)
- Open `test-report.html` in browser

## Troubleshooting

### "I don't see View/Download buttons!"

**Possible reasons:**
1. ❌ **Docker not running** → Start with `./manage.sh start`
2. ❌ **Tests didn't run** → Check if you clicked "Build & Run Tests"
3. ❌ **Frontend not updated** → Refresh browser (Ctrl+F5)

**Solution:**
1. Start Docker: `./manage.sh start`
2. Make sure all services running: `./manage.sh status`
3. Refresh browser page completely
4. Run tests again

### "Reports page is empty!"

**This means:**
- No test reports have been generated yet
- Reports are only created AFTER running tests

**Solution:**
1. Go to Generate page
2. Generate tests
3. Click "Build & Run Tests"
4. Report will auto-generate
5. Now check Reports page again

### "I found test-report.html but it won't open!"

**Try:**
1. Right-click the file
2. "Open with" → Choose your browser (Chrome, Firefox, Edge)
3. Or drag-and-drop file into browser window

- Total tests run
- Tests passed
- Number of assertions
- Exit code

- Complete compilation logs
- Shows all files compiled
- Any warnings or errors

### Test Results
- Test execution output
- Pass/fail for each test
- Detailed error messages if any

### Footer
- Professional branding
- Generated by CppUTest RAG Generator
